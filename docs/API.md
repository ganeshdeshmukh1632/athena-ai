# API Reference

Base URL (production): `https://athena-ganesh.duckdns.org/api`
Interactive docs: `/docs` (Swagger UI, auto-generated by FastAPI)

## Auth
- `POST /api/v1/auth/register` — `{email, password}` → `{access_token, token_type}`
- `POST /api/v1/auth/login` — `{email, password}` → `{access_token, token_type}`

All endpoints below require `Authorization: Bearer <token>`.

## Analysis
- `POST /api/v1/analyze` — `{question}` → full AnalyzeResponse (summary, evidence, risks, confidence, sources, technical/fundamental/risk/news breakdowns)
- `GET /api/v1/history?limit=20` — list of the current user's past analyses

## Watchlist
- `GET /api/v1/watchlist` — list of the current user's watchlist items with live snapshots
- `POST /api/v1/watchlist` — `{symbol}` → adds a symbol (must be a known Nifty 50/Next 50 ticker)
- `DELETE /api/v1/watchlist/{id}` — removes an item

## Misc
- `GET /health` — liveness check, no auth required
